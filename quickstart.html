Quick start

1. git clone example-minimal

```sh git clone git@github.com:ebean-orm-examples/example-minimal.git ```

2. mvn clean test

```sh cd example-minimal mvn clean test ```

3. Install IDE enhancement plugin

IntelliJ plugin

Installing and using the enhancement plugin for IntelliJ IDEA.

Eclipse plugin

Installing the enhancement plugin for Eclipse IDE.

4. Play around

Modify the minimal application adding entities, properties, tests etc.

Create a new project via archetype

```sh mvn archetype:generate \ -DinteractiveMode=f \ -DarchetypeGroupId=org.avaje.ebean \ -DarchetypeArtifactId=archetype-start \ -DarchetypeVersion=1.0 \ -DgroupId=com.foo.myapp \ -DartifactId=my-app ```
