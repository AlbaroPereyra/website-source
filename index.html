About Ebean

JPA Mapping

Ebean uses JPA mapping annotations.

```java @Entity @Table(name="customer") public class Customer extends BaseModel { @Column(length=100) String name; @ManyToOne(cascade=CascadeType.ALL) Address billingAddress; @OneToMany(mappedBy="customer") List contacts; ... ```

Type safe queries

Query beans

Use Query beans to build queries in a type safe manner.

```java LocalDate lastWeek = ...; List customers = new QCustomer() .name.istartsWith("rob") .billingAddress.city.equalTo("Auckland") .status.isEqualTo(Status.NEW) .registered.before(lastWeek) .setMaxRows(10) .orderBy() .name.asc() .id.desc() .findList(); ```
