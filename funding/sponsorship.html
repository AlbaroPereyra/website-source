Sponsored features

Imilia.com

2009 - Transparent encryption

The @DbEncrypt feature allows for transparent encryption of specific entity properties in the database meaning that data at rest is encrypted.

This feature uses database platform specific functions to encrypt and decrypt the data as needed and provide developers with a seamless approach to using encryption in the database.

The development of this feature was funded by imilia.com in 2009.
