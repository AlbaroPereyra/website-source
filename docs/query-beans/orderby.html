OrderBy

Properties have asc() and desc() such that they are added to the orderBy clause.

```java List rob = new QCustomer() .status.in(Status.NEW, Status.ACTIVE) .orderBy() .name.desc() // order by t0.name desc .findList(); ```

Produces the SQL below:

```sql select ... from customer t0 where t0.status in (?, ?) order by t0.name desc ```

We can add multiple properties to the orderBy clause.

```java List customers = new QCustomer() .status.in(Status.NEW, Status.ACTIVE) .orderBy() .name.desc() .id.asc() .findList(); ```

OrderBy expression

We can alternatively specify an order by expression which can include sql formula via orderBy(string).

```java List rob = new QCustomer() .status.in(Status.NEW, Status.ACTIVE) .orderBy("coalesce(activeDate, now()) desc, name, id") .findList(); ``` <@next "Queries" "/docs/query"/> <@edit "/docs/query-beans/orderby"/>
