(Optional but recommended)

The command line tool is optional, we don't strictly need it but it does some useful tasks. We can do all these tasks manually.

  • Generates Finders
  • Generates ebean.mf
  • Add GenerateDbMigration (code to generate DB Migrations)
  • Add application-test.yml
  • Add logback-test.xml
