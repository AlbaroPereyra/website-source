Add dependencies

Main dependency

Add the ebean dependency.

<@maven "io.ebean" "ebean" "auto" "3" />

Query beans dependency

Optionally add the ebean-querybean dependency. This is only needed when we want to use query beans to write type safe queries.

<@maven "io.ebean" "ebean-querybean" "auto" "3" />

JDBC Driver(s)

Add the dependency for the JDBC Driver(s) you want to use.

  • H2
  • Postgres
  • MySql
  • SQL Server
  • Oracle
```text com.h2database : h2 : 1.4.191 ```
```text org.postgresql : postgresql : 42.2.2 ```
```text mysql : mysql-connector-java : 8.0.11 ```
```text com.microsoft.sqlserver : mssql-jdbc : 6.4.0.jre8 ```
```text com.oracle.jdbc : ojdbc8 : 12.2.0.1 ```

Test Dependency

Add the following test dependency.

<@maven "io.ebean.test" "ebean-test-config" "auto" "3" />
