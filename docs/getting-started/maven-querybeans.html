APT - Generate query beans

We can optionally add a Java annotation processor (APT) to generate query beans. This allows us to write type safe queries.

Skip this if you don't want to use query beans.

Add ebean-querybean dependency

<@maven "io.ebean" "ebean-querybean" "auto" "3" />

Java APT

To generate Java query beans with maven add provided scope dependency for querybean-generator

```xml io.ebean querybean-generator 11.32.1 provided ```

Kotlin KAPT

To generate Kotlin query ebeans we need to register the kotlin-querybean-generator with the kotlin compiler. The easiest way to do this is to use a maven tile.

Add the maven tile into the build / plugins section of the pom.xml.

This performs enhancement of the entity beans, @Transactional methods and query beans at build time. 


  io.repaint.maven
  tiles-maven-plugin
  2.12
  true
  
    
      
      io.ebean.tile:enhancement:7.4
