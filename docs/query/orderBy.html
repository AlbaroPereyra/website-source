orderBy (Control sort order)

Control the ordercode> of your SQL query. There are several methods available:

  • orderBy(String clause)
  • order(String clause)
  • orderBy()
  • order()

orderBy(String clause) and order(String clause) are equivalent to each other and both receive the string of comma separated properties and sort direction (much like SQL code), such as name asc, status desc

```java Ebean.find(Customer.class) .orderBy("name asc,status desc") .findList() ```

orderBy() and order() are also equivalent and allow you to use a fluent interface to adding sort properties. See the example:

```java Ebean.find(Customer.class) .order() .asc("name") .desc("status") .findList() ```
